1. Quick start

  1. Complete or skip the interactive review demo. The demo explains question, answer, and rating actions with real cards.
  2. On Import Your Decks, choose an .apkg file, restore from your iCloud if this is a new device, or continue and import later.
  3. Review the detected deck, card, media, and review-history totals before writing anything to your library.
  4. Open Today, select a study scope, and start the session.

The demo deck is optional practice material, does not count toward the Free card allowance, and can be deleted from Decks.

2. How content is organized

Folders

A Folder is your personal organization layer. It contains decks, not individual cards. Folders can be nested up to three levels and do not modify the original deck names inside an imported package.

Deck hierarchy

Deck names that contain :: are displayed as a collapsible tree. For example, Biology::Genetics::Chapter 5 appears under Biology → Genetics → Chapter 5. Selecting a parent level includes all real decks below it.

Cards

Every card belongs to one deck. Moving a deck to another Folder changes only your Remin organization; it does not change card ownership or break the imported deck hierarchy.

3. Import decks

  1. In Decks, select the add button and choose Import Anki .apkg.
  2. Select a file from Files.
  3. Inspect package warnings, deck count, card count, media, templates, and review history.
  4. Choose a Folder or leave the decks ungrouped.
  5. Choose whether to keep supported scheduling and review progress, then start the import.

The progress window prioritizes card progress and also shows completed and remaining decks. Import History records the result and the first imported deck names so you can identify previous operations.

Large packages

Keep the app in the foreground and connect power for very large media packages. Remin Cards performs an incremental merge using Anki identifiers where available, but always keep the original .apkg until you verify the result.

4. Choose what to study

From Today, choose your study scope in two stages:

  1. Optionally choose a Folder. The Folder picker opens as a collapsible tree so it does not expand the Today page.
  2. Choose any deck-tree level. A top-level or intermediate choice includes every child deck; a leaf choice includes only that real deck.

Remin Cards remembers your last scope. The deck tree starts collapsed to its first level, and you can use search when the full deck name is easier to find by text.

Study history

Today groups learning history by the top-level deck name, not by Folder. Each row can show today’s studied cards, progress through the selected content, and mastery-oriented progress without truncating key values.

5. Review cards

  1. Read the question. Tap the main card area or select Show Answer.
  2. Review the answer, then choose Again, Hard, Good, or Easy.
  3. The rating is recorded and the next card appears. The interval shown above each button is a forecast calculated from the card state and current preset.

Use the A− and A+ controls to change content size. Reset returns to the default percentage, and the question or answer is recentered after a scale change.

Card actions

  • Undo: restore the most recently rated card and its review entry.
  • Flag/importance: mark content that deserves attention.
  • Skip today: bury the card until the next day.
  • Suspend: remove low-quality or unwanted content from future study queues while keeping the card and history.

6. FSRS and study presets

FSRS uses review history to estimate memory stability and difficulty. In Settings → Study Algorithm → FSRS Preferences, set desired retention from 80% to 100%. Higher retention schedules more frequent reviews.

Use Optimize periodically when enough real review history exists. Optimization creates parameters fitted to your reviews; running it repeatedly without meaningful new history is unnecessary.

Study presets also control daily new/review limits, learning and relearning steps, display order, sibling burying, leech behavior, and maximum interval. A deck can use a preset without changing the imported card content.

For most users, start near 90% desired retention. Raising it toward 100% can increase workload sharply.

7. Create and edit cards

Open Create for a new card, or select a card from deck details or the Card Browser to edit it. Choose the destination deck first; cards cannot belong directly to a Folder.

The Anki-style field editor supports:

  • multiple named fields and note types;
  • bold, italic, underline, superscript, subscript, lists, alignment, color, and clear formatting;
  • cloze syntax, source/HTML editing, image attachments, and recorded audio;
  • tags, flags, suspension, and card preview.

Toolbar buttons show hover help on pointer platforms. Preview before saving complex HTML or media-heavy changes.

8. Card Browser and statistics

Open the Card Browser from the Decks page menu. Search card text and combine filters for Folder, deck, tags, state, importance, and sort order. Selecting a card opens its details and editor.

Advanced Statistics summarizes review activity, retention, due trends, card states, and deck distribution. Exam Plan appears on Today so upcoming work can be translated into a practical daily workload.

9. Sync with your iCloud

Open Settings → Data & Sync → Sync Now to My iCloud. The sheet reports completed records, waiting records, progress, and errors. Sync is local-first: an action is saved on the device before it is sent to iCloud.

For a new device, you can choose iCloud restore from the onboarding bridge. Use the same Apple Account and allow the initial restore to finish before making large edits on both devices.

Do not edit the same card simultaneously on multiple offline devices. If a conflict is detected, Remin Cards preserves information needed to resolve it rather than silently treating every change as identical.

10. Export your decks

Open Settings → Export Your Decks.

  • .apkg: choose a Folder or any deck-tree level. The export includes selected deck hierarchy, supported templates, media, scheduling, and review history. Progress shows preparation, cards, media, and completion. On Mac, choose the destination folder.
  • CSV: use for spreadsheet editing or text analysis. CSV does not preserve media, templates, scheduling, or review history.

Exported .apkg filenames include date and time down to seconds to avoid accidental overwrites.

11. Free and Pro

Free supports up to 1,000 non-suspended active cards. Demo cards do not count; suspending cards or deleting decks releases capacity. You can inspect a larger import before choosing to upgrade.

Remin Cards Pro unlocks unlimited active cards, full iCloud sync, Advanced Statistics, advanced Card Browser filters, and .apkg export. Pro is offered as an annual auto-renewable subscription and a one-time Lifetime purchase. Use Restore Purchases after reinstalling or moving to another device.

12. Safe data habits

  • Keep original .apkg files until you verify imported media and scheduling.
  • Export important decks before deleting a hierarchy or clearing local data.
  • Remember that deleting local app data does not necessarily delete iCloud data.
  • Import custom JavaScript templates only from sources you trust.
  • Never post private or copyrighted deck files in a public support issue.

1. 快速开始

  1. 完成或跳过互动复习演示。演示使用真实卡片说明查看问题、显示答案和评分。
  2. 在“导入你的牌组”中选择 .apkg；若这是新设备，也可从自己的 iCloud 恢复；还可以稍后再导入。
  3. 写入资料库前，确认识别到的牌组、卡片、媒体和复习历史数量。
  4. 进入“今天”，选择学习范围并开始复习。

演示牌组只是可选练习材料，不计入 Free 卡片额度，可从“牌组”页面删除。

2. 内容层级

我的分组

“我的分组”是你的个人整理层，只管理牌组，不直接包含卡片。分组最多支持三级，不会修改导入包中的原始牌组名。

牌组树

包含 :: 的牌组名会显示成可折叠树。例如 生物::遗传::第 5 章 显示为“生物 → 遗传 → 第 5 章”。选择任意父级会包含其下所有真实牌组。

卡片

每张卡片只属于一个牌组。把牌组移到另一个“我的分组”，只会改变 Remin 中的个人整理方式，不会改变卡片归属或破坏导入的牌组层级。

3. 导入牌组

  1. 在“牌组”页面点击添加，选择“导入 Anki .apkg”。
  2. 从“文件”中选择牌组包。
  3. 检查警告、牌组数、卡片数、媒体、模板和复习历史。
  4. 选择“我的分组”，或保留为未分组。
  5. 选择是否保留支持的调度与复习进度，然后开始导入。

进度窗口以卡片进度为主，同时展示已完成和剩余牌组。导入历史会保存结果和前几个牌组名称，便于识别以前的操作。

大型牌组包

导入大量媒体时，让 App 保持前台并连接电源。Remin Cards 会尽可能利用 Anki 标识符进行增量合并，但在确认导入结果前，请保留原始 .apkg。

4. 选择学习范围

在“今天”中分两步选择：

  1. 可选“我的分组”。分组通过弹窗中的折叠树选择，不会撑长“今天”页面。
  2. 选择任意牌组树层级。首级或中间层会包含全部子牌组，叶子级只包含该真实牌组。

Remin Cards 会保留上一次选择。牌组树默认只显示首级，可逐层展开；也可以通过完整牌组名模糊搜索。

学习历史

“今天”的学习历史按首级牌组名归集，而不是按“我的分组”。每行可展示今日已学、选定内容的总学习情况以及掌握进度等关键指标。

5. 复习卡片

  1. 阅读问题，轻点主要卡片区域或“显示答案”。
  2. 查看答案后，选择“重来”“困难”“良好”或“简单”。
  3. 评分会写入复习历史，然后显示下一张卡片。按钮上方的时间是根据卡片状态和当前预设计算的预计间隔。

使用 A− 和 A+ 调整内容字体，复位按钮恢复默认百分比。缩放后问题或答案会重新居中。

卡片操作

  • 撤销：恢复最近一次评分前的卡片及复习记录。
  • 标记/重要性：标记需要额外关注的内容。
  • 今天跳过：将卡片埋藏到第二天。
  • 暂停：让低质量或暂时不需要的内容退出学习队列，同时保留卡片和历史。

6. FSRS 与学习预设

FSRS 根据复习历史估计记忆稳定性和难度。在“设置 → 学习算法 → FSRS 偏好设置”中，可将期望留存率设为 80%–100%。留存率越高，复习越频繁。

积累足够真实复习历史后，可定期点击“优化”生成适合个人的参数。没有新增有效历史时，无需频繁重复优化。

学习预设还包括新卡/复习每日上限、学习与重学步骤、显示顺序、兄弟卡埋藏、Leech 处理和最大间隔。给牌组更换预设不会修改卡片内容。

多数用户建议先从约 90% 期望留存率开始。接近 100% 时，复习量可能显著增加。

7. 新建与编辑卡片

从“制卡”新建卡片，或从牌组详情/卡片浏览器点开卡片进行编辑。先选择目标牌组；卡片不能直接属于“我的分组”。

Anki 风格字段编辑器支持：

  • 多个命名字段和笔记类型；
  • 粗体、斜体、下划线、上下标、列表、对齐、颜色和清除格式；
  • 填空语法、源码/HTML、图片附件和录制音频；
  • 标签、标记、暂停和卡片预览。

在鼠标平台上，工具栏按钮提供悬浮说明。保存复杂 HTML 或大量媒体修改前，建议先预览。

8. 卡片浏览器与统计

从牌组页菜单打开卡片浏览器。可搜索卡片文字，并组合“我的分组”、牌组、标签、状态、重要性和排序等筛选。点击卡片可打开详情与编辑器。

高级统计汇总复习活动、留存率、到期趋势、卡片状态和牌组分布。“考试计划”位于“今天”，可将目标转化为可执行的每日学习量。

9. 同步到你的 iCloud

进入“设置 → 数据与同步 → 立即同步到我的 iCloud”。弹窗会显示已完成记录、等待记录、进度和错误。同步采用本地优先方式：操作先保存在设备，再发送到 iCloud。

在新设备首次引导中，可以选择从 iCloud 恢复。请登录同一 Apple 账户，并让首次恢复完成后，再同时在多设备进行大量编辑。

避免在多台离线设备上同时编辑同一张卡。检测到冲突时，Remin Cards 会保留解决冲突所需的信息，而不是直接把所有改动视为相同。

10. 导出你的牌组

进入“设置 → 导出你的牌组”。

  • .apkg：选择“我的分组”或任意牌组树层级。导出包含选定牌组层级、支持的模板、媒体、调度和复习历史。进度会展示准备、卡片、媒体和完成阶段；Mac 上可以选择保存文件夹。
  • CSV：适合电子表格整理或文本分析，不保留媒体、模板、调度和复习历史。

导出的 .apkg 文件名包含精确到秒的日期时间，减少意外覆盖。

11. Free 与 Pro

Free 最多支持 1,000 张非暂停 active cards。演示卡片不计入；暂停卡片或删除牌组会释放额度。超额牌组可以先检查内容，再决定是否升级。

Remin Cards Pro 解锁无限 active cards、完整 iCloud 同步、高级统计、卡片浏览器高级筛选和 .apkg 导出。Pro 提供年度自动续订和一次性永久版；重装或更换设备后使用“恢复购买”。

12. 数据安全习惯

  • 确认媒体和调度正确导入前，保留原始 .apkg。
  • 删除牌组层级或清除本地数据前，导出重要牌组。
  • 清除本地 App 数据不一定会删除 iCloud 数据。
  • 只导入可信来源的自定义 JavaScript 模板。
  • 不要在公开支持 Issue 中发布私人或受版权保护的牌组。
