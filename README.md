# Remin Cards Support

Public support site for Remin Cards, including:

- Product and contact page
- Privacy Policy
- Support and troubleshooting
- User Guide
- Terms of Use

The site is plain static HTML/CSS/JavaScript and is ready for GitHub Pages.

## Public URLs

After GitHub Pages is enabled for the repository root:

- Home: `https://kevincy.github.io/remin-support-public/`
- Privacy: `https://kevincy.github.io/remin-support-public/privacy.html`
- Support: `https://kevincy.github.io/remin-support-public/support.html`
- User Guide: `https://kevincy.github.io/remin-support-public/guide.html`
- Terms: `https://kevincy.github.io/remin-support-public/terms.html`

## Local preview

Run any static HTTP server from this directory, for example:

```sh
python3 -m http.server 8080
```

Then open `http://localhost:8080`.
